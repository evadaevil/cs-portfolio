What is HTML and CSS?

HTML is important because it is the language that most websites are made with.
HTML is the basic outline of websites. In 1980 a physicist named
Tim Berners-Lee and a contractor from CERN made a prototype of ENQUIRE
so they could share their documents; Then later developed HTML in the
late 1990s. HTML is started with "" and has multiple tags.
CSS is the design of the website (font color, background, ect.)
CSS is written to describe the presentation of the HTML code.
A tag is what you use in programming so the computer knows what you are
programming. To write a tag add "<>" with whatever function you want in between the tags.
An attribute is a value which is between the "<>" tags. If you change the attribute, you change the
function of it. For example you change < title > < /title > tags to < h1 > < / h1>, the title
becomes a heading.
